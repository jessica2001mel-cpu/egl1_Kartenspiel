# egl1_Kartenspiel: THE DECK OF FOUR TALES
<img width="3024" height="4032" alt="IMG_3535" src="https://github.com/user-attachments/assets/3af18615-10c3-4c73-9b2e-7287abf89a31" />

**Gestaltung eines klassischen Kartenspieles mit animierten Effekten (Daumenkino / stop motion style)**

**Idee:**
- Jede Karte hat feste Elemente (Herz, Karo, Kreuz, Pik, Dame, König, Bube,  Joker, Ass) 
- Bestimmte Elemente bewegen sich durch das blättern der Karten (Daumenkino Style)

**Rahmen des Kartenspieles:**
- 55 Karten
- Kartenbox
  
**Aufteilung:**
- 1x Deckkarte 
- 9x Zahlenkarten (2 bis 10) (Herz, Karo, Kreuz, Pik) -> insgesamt 36 Zahlenkarten 
- 12x Bildkarten (jeweils 4x Bube, Dame und König) 
 -4x Asse 
 -2x Joker

**Animation:** 
Illustrationsstil im fineline / handdrawn style 
- Animation wird aufgeteilt in die verschiedenen Symbole (Herz, Karo, Kreuz und Pik)
- Karten können in der richtigen Reihenfolge wie bei einem Daumenspiel die Animation bewegen 

*Symbole:*
- **Herz**  = Zwei Menschen kommen aufeinander zu und küssen sich Herz 
- **Pik**  = Sanduhr die sich dreht  
- **Karo**  = Schatztruhe geht auf 
- **Kreuz** = Zwei Schwerter kreuzen sich

*Bedeutung:*
THE DECK OF FOUR TALES: Jedes einzelen Symbol hat seine eigene Geschichte
